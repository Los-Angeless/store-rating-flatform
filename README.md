# Store Rating Platform

## Overview
Store Rating Platform is a web application that allows users to rate and review stores. 

## Features
- User signup/login
- Store ratings and reviews
- Search and filter stores
- Admin dashboard 

## Tech Stack
- Frontend: React.js
- Backend: Node.js / NestJS 
- Database: MySQL
